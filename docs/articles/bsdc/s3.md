
## Why not much activity in this space?

- Compilers are generally closed systems
- Few offer type system hooks
- Fewer support AST mods
- javac is no exception
- javac metaprogramming APIs: 
  - annotation processors
  - javac plugins
  - AST is _**read-only**_
  - No type system hooks
