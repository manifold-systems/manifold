
### Why don't we see more compile-time metaprogramming in Java?  
- Compilers are generally closed systems
- Few support AST mods (forbid modifying user source files)
- Fewer offer extension points (e.g. can't override the type resolver)
- javac is no exception
- Compile-time metaprogramming is limited to: 
  - annotation processors
  - javac plugins 
  - Official APIs limit AST access to read-only
