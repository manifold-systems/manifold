

## Manifold
- javac plugin
- A foundation for compile-time metaprogramming
- Provides features built on this foundation
- Uses internal compiler APIs 
- Modifies ASTs
- Hooks into compiler pipeline 
- Single binary supports JDKs 8 - 26 (LTS releases + latest)
