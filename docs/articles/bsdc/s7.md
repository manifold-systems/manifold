## Type manifolds
- Type-safe: schemas, queries, templates, markup, DSLs, etc.
  - [SQL](https://github.com/manifold-systems/manifold/tree/master/manifold-deps-parent/manifold-sql/readme.md), [GraphQL](https://github.com/manifold-systems/manifold/tree/master/manifold-deps-parent/manifold-graphql)
  - [JSON & JSON Schema](https://github.com/manifold-systems/manifold/tree/master/manifold-deps-parent/manifold-json)
  - [XML](https://github.com/manifold-systems/manifold/tree/master/manifold-deps-parent/manifold-xml), [YAML](https://github.com/manifold-systems/manifold/tree/master/manifold-deps-parent/manifold-yaml)
  - [Java itself!](https://github.com/manifold-systems/manifold/tree/master/manifold-deps-parent/manifold-ext)
- Two kinds of type manifolds
  - *Primary*: provide the main type
  - *Supplementary*: augment the type (methods, interceptors, etc.)
