## Manifold core
- **Two primary APIs**:
- *Type Manifolds* - API for *Just-in-time* type projection
  - Override javac's type system
  - Claim or ignore type names
  - Project from: resources, URLs, anything
  - Types are lazy virtual source files (no actual file i/o)
- *Compiler Components* - API for language enhancements
  - Add new language constructs (Parser mods + AST mods)
  - Augment existing language features (AST mods)
