## Manifold core
- A set of APIs for compile-time metaprogramming
- *Just-in-time* type provider API
  - As javac resolves type names providers are consulted first
  - Can choose to claim or ignore the type name
  - Types determined from resources (files, URLs, anything)
  - Providers project source directly into javac (no file i/o)
- Compiler component API
  - For adding/modifying *language features*
