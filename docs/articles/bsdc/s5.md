## Architecture
- Monolithic, multi-module project
- Core manifold modules: `manifold`, `manifold-rt`
- Feature modules built atop core follow this pattern:<sub>
  - `manifold-xxx` -- exclusive to compile-time
  - `manifold-xxx-rt` --  runtime, manifold-xxx dependency
  - `manifold-xxx-test` --  runtime tests</sub>
- Use features in your project (Gradle): <sub>
  - implementation `'systems.manifold:manifold-xxx-rt:2026.1.8'`
  - annotationProcessor `'systems.manifold:manifold-xxx:2026.1.8'`
  - testImplementation `'systems.manifold:manifold-xxx-rt:2026.1.8'`
  - testAnnotationProcessor `'systems.manifold:manifold-xxx:2026.1.8'`
  </sub>
