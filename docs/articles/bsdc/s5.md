## Architecture
- Monolithic codebase
- Core manifold modules: `manifold`, `manifold-rt`
- Plus feature modules built atop core:
  - `manifold-ext` (extension methods, iterceptors, etc.)
  - `manifold-json` (JSON/JSON Schema support)
  - `manifold-sql` (type-safe/inline SQL)
  - etc. 
- Feature modules are separate dependencies
- Maven build
