## IDE support

- Manifold supports IntelliJ IDEA (plugin)
- Type Manifolds: Plug-and-play
  - Plugin detects TM libraries in project modules
  - Drives TMs to project code (lazy, per type)
- Language features
  - _Not_ plug-and-play
  - Independently integrated
