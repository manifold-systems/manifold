## IDE support
- Compiler features must be mirrored in the IDE
- Manifold supports IntelliJ IDEA (plugin)
- Type Providers: Plug-and-play
  - Plugin detects type provider libraries used in project modules
  - Plugin drives type providers to generate code
  - Annotations in API provide hints
- Language features: not so much
  - Requires deep integration
  - IntelliJ platform API is well architected
