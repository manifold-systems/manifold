
## Compile-time metaprogramming
- Developer _**productivity**_
- Comparable to runtime metaprogramming *(given compiler APIs)*
- Dev-time _**determinism**_
  - compile errors, not runtime errors
  - code completion
  - usage searching
  - refactoring
- Incremental compilation
  - resource files are virtual source files
